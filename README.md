# Fouraxes generator
Battle the web with Backbone, Handlebars, RequireJS and Harmony!

## Subgenerators
The Fouraxes generator contains the following subgenerators.

```bash
yo fouraxes:model ...
yo fouraxes:view ...
yo fouraxes:collection ...
yo fouraxes:router ...
yo fouraxes:template ...
```
